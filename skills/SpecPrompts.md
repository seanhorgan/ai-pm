

# 1 pager

This 1-pager should include the following elements
- Crisp statement that covers why this work is important (3 sentences max). 
- Metric that measure the business impact (revenue/cost/etc) and how we will measure them. If those aren't clear, propose and state assumptions for what they could be. 
- Clear list of "Jobs To Be Done" (JTBD). At Google we use the term "CUJ" which is a Critical User Journey. The idea is to articulate the problems you're solving from the user's perspective. For each CUJ list the user role, their goal, and the pain points you're addressing.
- Detailed specifications that outline the technical requirements for the project.
- Include a References section that includes links to each of the attached artifacts and links considered in this specification
